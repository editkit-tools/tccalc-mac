# TC Calc — Mac Releases

Sparkle update feed and release assets for **TC Calc** (macOS).

- **Appcast:** `https://editkit-tools.github.io/tccalc-mac/appcast.xml`
- **App:** [editkit.tools/tools/tccalc.html](https://editkit.tools/tools/tccalc.html)

Each release contains `TCCalc.dmg`. Release notes in `release-notes/` are served via GitHub Pages and loaded by the Sparkle update dialog.
